# 📊 Simulador de Produção Industrial em Tempo Real

Este projeto simula dados de uma fábrica, utilizando Python para geração dos dados, PostgreSQL para armazenamento e Power BI para visualização.

## 🚀 Objetivo
Demonstrar a integração entre coleta de dados, banco de dados e ferramentas de Business Intelligence.

## 🧠 Funcionalidades
- Simulação de produção e temperatura
- Armazenamento em PostgreSQL
- Atualização contínua dos dados
- Dashboard interativo no Power BI

## 🛠️ Tecnologias
- Python
- PostgreSQL
- SQL
- Power BI

## 📈 Dashboard
- Produção por minuto (peças/min)
- Temperatura por minuto (°C)

## ▶️ Como executar
1. Instale dependências:
   pip install psycopg2

2. Configure o PostgreSQL

3. Execute:
   python simulador_fabrica.py

4. Conecte o Power BI ao banco

## 💡 Aprendizados
- Integração de sistemas
- Manipulação de dados
- Visualização com Power BI
